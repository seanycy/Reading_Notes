# 4.1 使用WHERE字句
在SQL语句中，只检索所需数据需要指定搜索条件（search criteria），即过滤条件（filter condition）。数据根据WHERE字句指定的搜索条件进行过滤。WHERE字句跟在表名（FROM字句）之后给出，在ORDER BY字句之前
```sql
SELECT prod_id,prod_price,pord_name
FROM products
WHERE prod_price = 3.49
```

# 4.2 WHERE子句操作符

|操作符|说明|操作符|说明|
|--------|-----|--|--|
|=|等于|>|大于|
|<>|不等于|>=|大于等于|
|!=|不等于|!>|不大于|
|<|小于|BETWEEN|在指定的两个值之间|
|<=|小于等于|IS NULL|为NULL|
|!<|不小于|
